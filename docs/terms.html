What the software does

ga-mcp-full is a Model Context Protocol server that runs locally on your machine and exposes Google Analytics 4 Admin API and Data API operations as tools usable by AI coding assistants. It is not a hosted service. The project maintainers do not operate a server that receives your data or credentials. See the Privacy Policy for details on local data handling.

  • You are responsible for complying with Google's API Terms of Service and the Google Analytics Terms of Service applicable to your account.

Acceptable use

Do not use ga-mcp-full to:

  • Access Google Analytics properties you are not authorized to access.

Abuse of the bundled OAuth client may result in Google revoking the client, which would interrupt access for all users. High-volume users are encouraged to configure their own OAuth client via the environment-variable override documented in the project README.
